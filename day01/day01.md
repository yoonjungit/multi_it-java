### 4. `<table>`

테이블 만들기

```html
<table> : 테이블 태그
<tr> : 테이블 행
<td> : 테이블 열

<!-- 칸 합치기 -->

<td colspan="3"> : 열 3칸 합치기
<td rowspan="3"> : 행 3칸 합치기
```
